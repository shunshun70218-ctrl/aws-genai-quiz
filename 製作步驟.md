# AWS GenAI 考試練習 — 製作步驟與流程

> 換機器時，把整個「aws 考試」資料夾複製過去即可。以下分兩種情境說明。

---

## 0. 資料夾內容一覽

| 檔案 | 說明 |
|------|------|
| `AWS_GenAI_Q1-Q97_題目格式.docx` | **原始** docx（內容曾被精簡/清理過，僅留存參考，勿當權威來源） |
| `questions_full.json` | ⭐ **唯一權威資料來源**。97 題的題目、選項、答案、考點背法，全部逐字照「貼上的原文」 |
| `AWS_GenAI_Quiz.html` | 產出物：可直接開的測驗網頁（自包含，含全部題目） |
| `AWS_GenAI_Q1-Q97_題目格式_修正版.docx` | 產出物：修正後的 Word（題目=權威原文 + 答案 + 考點背法） |
| `build_quiz.py` | 用 `questions_full.json` 重新產生測驗網頁 |
| `build_docx.py` | 用 `questions_full.json` 重新產生修正版 Word |
| `製作步驟.md` | 本文件 |

`questions_full.json` 每筆格式：
```json
{ "num": 1, "stem": "題幹…", "options": {"A":"…","B":"…","C":"…","D":"…"},
  "answers": ["D"], "explain": "考點背法說明…" }
```
- 共 97 題：86 題單選(4 選項) / 9 題複選兩個(5 選項) / 2 題複選三個(6 選項)。
- ⚠️ 刻意保留原文拼字，**不要更正**：`DynamoDPurchase`(#44B)、`PostgresQL`(#28C)、`PostgresSQL`(#47C)、`Sage Maker AI`(#78A) 等。

## 情境 A：只想在新機器「使用」（最簡單）

不需要 Python。直接複製這兩個檔過去即可：

1. **測驗網頁**：雙擊 `AWS_GenAI_Quiz.html` 用瀏覽器開即可作答。
   - （可選）若想用網址開：在資料夾內開終端機執行
     ```bash
     python3 -m http.server 8765
     ```
     再開瀏覽器到 `http://localhost:8765/AWS_GenAI_Quiz.html`
2. **Word**：用 Word／Pages 開 `AWS_GenAI_Q1-Q97_題目格式_修正版.docx`。

---

## 情境 B：想在新機器「重新產生 / 修改」題目

### 環境需求
```bash
python3 --version          # 需 Python 3
pip install python-docx    # 產生 Word 時需要（產生網頁不需要）
```

### 改題目
編輯 `questions_full.json`（唯一來源），改完後重新產生：
```bash
cd "aws 考試"          # 進到資料夾
python3 build_quiz.py   # → 重新產生 AWS_GenAI_Quiz.html
python3 build_docx.py   # → 重新產生 AWS_GenAI_Q1-Q97_題目格式_修正版.docx
```
兩個產出都會直接覆蓋同名檔。

## 測驗網頁功能（已內建）

- 每次隨機抽題、打亂順序（可在開始畫面設定出題數量，最多 97）。
- 自動辨識單選/複選；複選題標示「請選 N 個」。
- 作答後即時顯示對錯、正確答案、考點背法。
- 每題上方標示「📄 原文件 Question #N」對應原始題號。
- 結束時統計正確率（圓環）、答對/答錯數、完整錯題回顧，並可「只練習錯題」。

## 當初的完整製作流程（紀錄，供回溯）

1. **取得題庫**：原始 `AWS_GenAI_Q1-Q97_題目格式.docx`（解壓 `word/document.xml`，抓 `<w:t>` 文字）。
2. **發現問題**：原始 docx 的多題題幹被精簡，且部分拼字被「自動更正」，與真正考題原文不符。
3. **以權威原文為準**：改用使用者貼上的完整原文（ExamTopics 格式：`Question #N` + 題幹 + 選項 A–F）。
4. **解析合併**：
   - 從權威原文解析出每題的 `stem` 與 `options`。
   - 從原始 docx 解析出 `answers`（`answer:` 行）與 `explain`（`考點背法` 段落）。
   - 以題號合併 → `questions_full.json`。
5. **驗證**：97 題齊全；每個正確答案字母都存在於該題選項；選項數/答案數分佈一致（86/9/2）。
6. **產生產出物**：`build_quiz.py` → 網頁；`build_docx.py` → 修正版 Word。
7. **再驗證**：Word 與 `questions_full.json` 逐字比對 0 處不符。

> 唯一非逐字之處：原文少數彎引號 `'` 統一為直式 `'`，其餘文字完全相同。

## 小提醒
- 資料夾若出現 `~$...docx` 之類檔案，是 Word 開啟時的暫存鎖檔，可忽略或關閉 Word 後刪除。
- `AWS_GenAI_Quiz.html` 為自包含單檔，沒有外部相依，複製到任何機器都能用。
